# Changelog

## 1.7.2 — 2026-07-20

### 改进

- 与 Windows 热切换修复版本同步发布；macOS 安装、主题切换与自动更新行为保持不变。

## 1.7.1 — 2026-07-20

### 修复

- macOS 更新器兼容带毫秒和不带毫秒的 ISO 8601 发布日期，修复签名清单有效但提示“更新发布日期无效”的问题。
- macOS 安装器移除已废弃的 `presets/` 初始化调用，统一通过 schema 2 内置主题目录完成首次安装和升级。
- 增加当前签名更新源日期解析与旧安装路径回归测试。

## 1.7.0 — 2026-07-19

### 新增

- 管理器窗口和 macOS 顶部“皮肤”菜单新增软件内更新入口，启动后每天自动检查一次，也可随时手动检查。
- 更新清单使用 Ed25519 签名，并对 DMG 执行版本、HTTPS、大小和 SHA-256 校验。
- 已验证的 DMG 会自动挂载并启动自动更新模式，部署引擎、管理器和 Skill 后重新打开新版。
- 新增独立在线主题目录，可发现并原子安装兼容的 schema 2 官方主题，不需要升级整个应用。

### 改进

- 自动更新期间 Codex 可以保持运行，已存在的配置、当前主题和用户自建主题会保留。
- 更新检查支持 24 小时节流、离线错误状态、版本说明和安装进度反馈。

## 1.6.1 — 2026-07-19

### 改进

- macOS 顶部菜单栏入口由单独的调色盘图标改为“调色盘图标 + 皮肤”，更容易定位常驻管理器。
- 增加菜单栏文字入口回归检查，防止后续版本退回到不直观的纯图标。

## 1.6.0 — 2026-07-19

### 新增

- 管理器原生驻留 macOS 菜单栏，显示当前主题和实时连接状态，可直接快速切换、打开 Codex 或恢复完整窗口。
- 主窗口关闭后继续在后台刷新主题库、活动主题与注入器进程状态。

### 改进

- 主窗口和菜单栏共享同一 `ThemeStore`，切换进度与外部状态变化会同步更新。
- 初音未来主题改为跟随系统自动切换浅色、深色调色板。

## 1.5.0 — 2026-07-19

### 新增

- 内置 `codex-skin-theme-creator` Skill，可通过 Codex 对话生成或重做主题并自动加入用户主题库。
- 管理器“接入标准”页新增 Skill 安装状态、重新安装和目录入口。
- macOS 管理器附带无界面主题创建 CLI，与软件内创建器共用 schema 2、裁切、校验和原子替换核心。

### 改进

- 管理器监测用户主题库变化，Skill 或外部导入完成后自动刷新，无需重新打开应用。
- 安装器自动部署主题创建 Skill，并在升级时同步最新 Skill 文件。

## 1.4.0 — 2026-07-18

### 新增

- 新增“蕾姆 · 冰蓝夜庭”主题，以 `gpt-image-2` 高质量模式生成月夜宅邸、蓝玫瑰与冰晶场景。
- macOS Studio 新增原生创建主题向导，支持图片焦点、名称、ID、作者、描述、分类、浅色/暗色和三组主题色。
- macOS Studio 新增严格 schema 2 导入校验、同 ID 自定义主题替换确认和内置主题 ID 保护。
- 接入标准页内置主题目录、尺寸、字段和宠物层规则。

### 修复

- 旧“换一张图”入口恢复为 schema 2 标准 PNG 三文件流程。
- 角色主题聊天页统一复用主背景和 `72% center / cover` 构图，不再为个别主题增加独立白化阅读层。
- “蔚蓝档案 · 青春合影”聊天页使用适配五人合影的满高取景，修复最右侧人物被横向裁掉的问题。
- 设置页侧栏、内容卡片、输入控件、下拉菜单和文字统一继承当前主题变量，修复浅色主题出现深色卡片与低对比文字的问题。
- 注入器识别设置页及分类切换的加载状态，停留在设置页也可热切换主题，并继续保持宠物窗口显示。

### 改进

- 发布版本更新为 `1.4.0`，README、主题格式、平台说明和发行构建说明完成开源整理。
- Studio 源码按创建视图、主题包服务和主界面拆分编译。

## 1.3.2 — 2026-07-18

### 新增

- 新增“蔚蓝档案 · 青春合影”主题，通过 `gpt-image-2` 生成阿拜多斯对策委员会五人校园合影主图。
- 新增“卡提希娅 · 风栖海境”“芙宁娜 · 水色剧场”“流萤 · 星海微光”“Saber · 誓约胜利”和“明日香 · 红色黄昏”五套 `gpt-image-2` 角色主题。
- 首页、聊天、侧栏、插件/技能库、输入器、菜单和通知使用独立的蓝白学院模块样式。

### 修复

- 运行状态兼容旧状态文件，并在注入器存活时明确显示“运行中”，不再显示“未知”。
- 主题预览图加入内存缓存，页面切换不再反复从磁盘解码整套主图。
- 主题切换只标记当前处理项，其他“一键切换”按钮不再整页变灰。

- 初音未来与 KUN 黑金舞台的皮肤卡片改为纯主背景缩略图，不再混入旧版界面截图。
- 插件/技能库验证改查库页面容器，不再因该页面没有聊天输入框而误报失败。
- 芙宁娜和 Saber 主图以 `gpt-image-2` 高质量模式重新生成，强化官方角色特征、面部精度、服装细节、光影层次和 3:1 主题构图；聊天页加入各自的构图焦点与正文阅读层。
- 明日香主图重新生成，人物面部移至横向约 59% 并保留左侧第三新东京市场景；芙宁娜主图重新取景并减弱首屏与内容页的白色遮罩。

### 改进

- 应用、图形安装器与产品界面统一更名为“Codex 皮肤管理器”。
- 发布版本更新为 `1.3.2`，安装器会清理旧名称的应用入口。

- 月薪喵任务页恢复清晰的右侧人物侧景、运行状态与打卡标语，窄窗口会自动收起装饰

- 自定义图片自动中心裁切为 `2400x800`，同时生成 `1200x400` 预览；首页、任务、插件和技能页复用同一主图。
- 新增资源库页面语义识别，避免 Codex 更新 DOM 层级后重新出现黑色搜索背景。
